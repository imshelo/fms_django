Open your Terminal/Command Prompt window. (make sure to add "python" and "pip" in your environment variables)
Change the working directory to the extracted source code folder. i.e. cd C:\Users\Personal-23\Desktop\django_ams
Run the following commands:
make venv
pip install Django
pip install cryptography

pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
